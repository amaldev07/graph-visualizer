# Graph Visualizer

A simple Angular app to render graphs from adjacency-list input.

## Features

- Render nodes and edges visually
- Supports directed and undirected graphs
- Accepts text adjacency lists and LeetCode-style edge lists
- Interactive graph canvas powered by Cytoscape.js

## Input Examples

```txt
A: B C
B: D
C:
```

```txt
[[1,2],[1,3]]
```

## Run Locally

```bash
npm install
npm start
```

Open:

```txt
http://localhost:4200
```

## Build

```bash
npm run build
```
